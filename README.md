# 🌦️ Weather Dashboard

A sleek and responsive web application that provides real-time weather information for any city worldwide. Built using HTML5, CSS3, JavaScript, and a live Weather API.

🔗 **[Live Demo](https://vismayas05.github.io/weather-dashboard/)**

---

## ✨ Features

* **Real-time Data:** Get accurate, up-to-date temperature details instantly.
* **Detailed Metrics:** Displays critical weather conditions including Humidity and Wind Speed.
* **Responsive Design:** Fully optimized for smooth viewing on both desktop and mobile screens.
* **Dynamic UI:** Clean and modern interface built with a user-friendly layout.

## 🛠️ Tech Stack

* **HTML5** - Markup and structure.
* **CSS3** - Custom styling and layout design.
* **JavaScript (ES6)** - Fetching API data and dynamic DOM manipulation.
* **Weather API** - Source for real-time meteorological data.
